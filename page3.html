What is Screen Time?

Screen time refers to the amount spent using an electronic device with a display screen. This includes:

  • Smartphones
  • Computers
  • Televisions
  • Video Game Consoles (Playstation, Xbox, etc)
  • Tablets

Screen time has been incorporated in all aspects of modern life. The 2 common types of screen time include:

  • Educational Screen Time
  • Recreational Screen Time

Educational Screen Time

Educational screen time refers to the time spent on digital devices to gain knowledge, build skills, or complete academic work.
Today, educational screen time is not just school assignments, it includes many different types of sources, such as:

  1. Learning Through Educational Videos
    Students often learning through video platforms like Youtube, where they can find
    • Math tutorials
    • Science experiments
    • History documentaries
    • Language learning lessons
    • College lecture recordings
  2. Coding and Technical Skills
    For technical majors like Computer Science, screen time is essential, as they need to:
    • Write code
    • Test programs
    • Debug errors
    • Run simulations
  3. Online courses
    Students now take full classes online using
    • Google classroom
    • Zoom
    • College learning platforms

Recreational Screen Time

Recreational screen time refers to time spent on digital devices primarily for entertainment or socialising. Examples include:

  • Playing video games
  • Watching shows
  • Scrolling on social media
  • Talking with friends
  • Watching videos
